# ブラウザ版テトリス

SEGAアーケード版の操作感を意識して作った、シンプルなブラウザ版テトリスです。

HTML5 Canvas だけで動きます。

![スクリーンショット](./screenshots.jpg)

## 起動方法

`index.html` をブラウザで開いてください。

## 操作方法

| キー | 動作 |
| --- | --- |
| `Enter` | ゲーム開始 |
| `←` | 左移動 |
| `→` | 右移動 |
| `↑` または `Space` | 右回転 |
| `Z` | 左回転 |
| `↓` | ソフトドロップ |
| `S` | 一時停止 / 再開 |
| `F3` | 画面サイズ切り替え |

## キー設定

`index.html` の先頭付近にある `KEY_BINDINGS` を書き換えると、好きなキーに差し替えられます。

`key` は実際の入力値です。

`label` はゲーム画面の操作方法に表示する文字列です。

キーを変更すると、画面右側の操作説明も自動で更新されます。

## ルール

- 7種類のテトリミノが落ちてきます。
- 横一列がそろうとラインが消えます。
- 新しいピースの出現位置が埋まっているとゲームオーバーです。
- レベルは `4` ライン消すごとに `1` 上がります。
- レベル上限は `99` です。
- ソフトドロップは `1` マスごとに `1` 点入ります。
- ライン消去の得点は `消した本数 × 消した本数 × 100` です。
- 盤面を空にすると `5000` 点のオールクリアボーナスが入ります。

## 開始レベル

URL パラメータで開始レベルを指定できます。

```text
index.html?level=50
```

- 指定できる範囲は `1` から `99` です。
- 不正な値のときは `1` から始まります。

## 収録内容

- SEGA風の壁キック付き回転
- 効果音
- 画面サイズの切り替え
